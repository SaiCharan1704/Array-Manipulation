# Array Manipulation

## Objective
Write a Java program to perform basic array operations.

## Features
- Find the largest element in an array
- Find the smallest element in an array
- Reverse the array elements

## Concepts Used
- Arrays
- Loops
- Conditional Statements
- User Input using Scanner

## Technologies
- Java
- Eclipse IDE

## Sample Output

Largest Element: 40
Smallest Element: 5
Reversed Array: 15 40 5 25 10
